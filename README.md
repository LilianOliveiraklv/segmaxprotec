
SEGMAX PROTEC - Site pronto para GitHub Pages
============================================

Instruções rápidas para publicar:

1. Crie um repositório no GitHub com o nome: segmaxprotec.github.io
2. Faça upload de todos os arquivos desta pasta (index.html, style.css, CNAME, img/)
3. No GitHub: Settings → Pages → escolha Branch: main (root) e salve.
4. Verifique no painel de DNS do seu provedor (SuperDomínios) que os registros A apontam para os IPs do GitHub Pages:
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153
   E que exista um CNAME para www se desejar apontar o www.

Formulário de contato:
- O formulário atualmente NÃO está conectado. Quando você criar uma conta no Formspree e receber seu endpoint (ex: https://formspree.io/f/xxxx),
  abra index.html e substitua o atributo action="#" pelo seu endpoint.
- Alternativamente, mantenha o formulário e use o WhatsApp como contato principal.

Suporte:
Se precisar, posso enviar um passo-a-passo com capturas de tela para subir os arquivos no GitHub.
